# 02 — Archetypes & Trainee Categories

## The 9-dimensional signature

Every client is described by a combination of 9 orthogonal dimensions:

| Dimension | Values | Code prefix |
|---|---|---|
| Goal | fat_loss, muscle_gain, recomp, strength, general_health | 3-char |
| Somatotype | ectomorph, mesomorph, endomorph | 4-char |
| Experience | beginner, intermediate, advanced | 3-char |
| Age group | young (18-30), adult (31-45), middle (46+) | 4-char |
| Sex | male, female | M/F |
| Activity | sedentary, mostly_sedentary, lightly_active, highly_active | 3-char |
| Diet | omnivore, vegan | 4-char |
| Environment | home_bodyweight, home_gym, gym_full | HOM/HGY/GYM |
| Session | 30, 45, 60, 90 min | number |

Example: `MUS-ECTO-BEG-YOUN-M-LIG-OMNI-GYM-60`

**Combinatorial space:** 25,920 unique signatures.

## Somatotype classification (M&S)

Based on [muscleandstrength.com](https://www.muscleandstrength.com/articles/body-types-ectomorph-mesomorph-endomorph.html):

- **Ectomorph** — small frame, fast metabolism, lean, hard gainer.
- **Mesomorph** — athletic, rectangular, gains muscle easily.
- **Endomorph** — soft/round, stocky, gains fat easily, strong legs.

Inferred from BMI + body-fat %, refined by wrist circumference (frame
size) in borderline cases.

## Trainee categories (RippedBody 9 types)

Source: [goal-setting 1](https://rippedbody.com/goal-setting-1/),
[2](https://rippedbody.com/goal-setting-2/),
[3](https://rippedbody.com/goal-setting-3/)

| Category | Body fat (male) | Has muscle? | Strategy |
|---|---|---|---|
| Skinny | <10% | No | Bulk |
| Fat but muscled | 22-30% | Yes | Cut |
| Muscled, few lbs to lose | 15-22% | Yes | Cut |
| Shredded | <10% | Yes | Maintenance / slow bulk |
| Fat & weak | 23-30% | No | Cut + newbie gains |
| Obese | >30% | — | Habit change + cut |
| Skinny-fat | 15-22% | No | Recomp / gentle bulk |
| Purgatory | varies | varies | Cycle cut/bulk |
| New trainee, healthy | varies | No | Recomp |

Each category carries a summary, common pitfalls, and tailored
recommendations.

## Cut/Bulk decision boundaries

Source: [rippedbody.com/cut-or-bulk](https://rippedbody.com/cut-or-bulk/)

| Sex | End cut at | End bulk at |
|---|---|---|
| Male | ~10% BF | ~20% BF |
| Female | ~18% BF | ~28% BF |

(Add ~8% BF for women relative to men.)
